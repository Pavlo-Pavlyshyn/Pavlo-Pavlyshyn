## Хід роботи

1. У ході завдання з умовними операторами було внесено такі зніни:  
Для змінної **number** призначене число більше 15.  
Змінну **second_number** переведено в пусте значення.   
В список **first_array** додано 3 компоненти, щоб отримати 5 елементів з двох масивів.   
У масиві **second_array** прибрано 1 елемент, щоб кількість елементів дорівнювала двом.
```python
number = 17
second_number = None
first_array = [1,2,3]
second_array = [1,2]

if number > 15:
    print("1")

if first_array:
    print("2")

if len(second_array) == 2:
    print("3")

if len(first_array) + len(second_array) == 5:
    print("4")

if first_array and first_array[0] == 1:
    print("5")

if not second_number:
    print("6")
```

2. У завданні з циклом **while** створено змінну **i** зі значенням **0**. Наступним кроком завдання було створено сам цикл **while** з умовою доки **i** менше від **6**. В циклі втілено оператор **if** з умовою якщо **i** буде дорівнювати **3** використовуємо **continue**.    
**Continue** - пропускає вказану ітерацію і продовжує виконання циклу. [Завдання взято тут.](https://www.w3schools.com/python/python_challenges_while_loops.asp)
```python
i = 0

while i < 6:
   i += 1
   if i == 3:
      continue
   print(i)
```

3. В завданні над циклом **for** створено список із трьох фруктів. Використання циклу **for** дало можливість пройтись по кожному елементу масиву. Використано оператор **if** із вказаною умовою, якщо **i** дорівнює **"banana"** виконується **break**.  
**Break** - повністю виходить з циклу та ігнорує наступні ітерації. [Завдання взято тут.](https://www.w3schools.com/python/python_challenges_for_loops.asp)
```python
fruits = ["apple", "banana", "cherry"]

for i in fruits:
    print(i)
    if i == "banana":
       break
```
